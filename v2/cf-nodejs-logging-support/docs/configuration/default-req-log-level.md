# Set default request loging level
Change the default logging level for all request by setting the property "reqLoggingLevel": <level>. For example:
* "reqLoggingLevel": "warn"
* "reqLoggingLevel": "debug"

Example:
```ts
{
 "reqLoggingLevel": "info"
}
```
